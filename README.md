# GitGrade

**RepoEvaluator** is a web application that lets you evaluate any GitHub repository. It provides a **score**, a **summary**, and a **personalized roadmap** to improve the repository.

---

## Features

- Enter any public GitHub repository URL.
- Get an overall score out of 100.
- View a detailed breakdown of:
  - Documentation
  - Code Quality
  - Testing
  - Community
- Receive a summary of strengths and weaknesses.
- Get a roadmap with suggestions for improvement.

---

## Technologies Used

- Python 3
- Flask (backend)
- HTML, CSS, JavaScript (frontend)
- Bootstrap 5 (responsive design)
